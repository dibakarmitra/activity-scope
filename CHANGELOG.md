# Changelog

All notable changes to `dibakar/activity-scope` will be documented in this file.

## [1.0.0] - 2025-12-25

### Initial Release

- Comprehensive Activity logging for Laravel.
- Fluent `ActivityBuilder` with auto-actor resolution.
- Privacy-first approach with IP anonymization and sensitive key scrubbing.
- Modular traits: `HasActivities` and `LogsActivity`.
- Human-readable message generation via `MessageBuilder`.
- Support for Laravel 10, 11, and 12.
